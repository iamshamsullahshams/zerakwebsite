6-month roadmap

First 6 months of computers

Follow this roadmap in order. Each month builds on the last — from hardware basics to AI tools and career preparation.

M1
Foundation

Month 1 — Computer & IT fundamentals

Aligned with: CompTIA A+ Core 1 (1101)

▾

Core topics

CPU, RAM, storage Motherboard & ports Windows & Linux basics File systems Installing software Troubleshooting steps Keyboard shortcuts Backup & recovery

Certification aligned

CompTIA A+ objectives BIOS/UEFI settings Device Manager Safe hardware handling
🛠
Project: Build a fictional help desk — diagnose and document 5 common PC problems with step-by-step solutions.
By end of month 1: Confidently use and explain how a computer works from hardware to software. The foundation for all future learning.
M2
Networking

Month 2 — Networking & the internet

Aligned with: CompTIA Network+ · Cisco CCNA intro

▾

Core topics

IP addresses (IPv4/IPv6) DNS, DHCP, HTTP/S Routers & switches WiFi vs wired Firewalls & VPNs OSI model (7 layers) Subnetting basics Cybersecurity intro

Certification aligned

CompTIA Network+ Cisco Packet Tracer Ping / traceroute / ipconfig Ports & protocols
🛠
Project: Design a home/office network in Cisco Packet Tracer — connect devices, assign IP addresses, test connectivity.
By end of month 2: Understand how the internet and local networks work. Required knowledge for almost every IT job.
M3
Programming

Month 3 — Programming fundamentals

Aligned with: PCEP Python Institute · Harvard CS50

▾

Core topics

Variables & data types If / else / elif Loops (for, while) Functions Lists & dictionaries File input/output Reading errors Algorithmic thinking

Certification aligned

PCEP exam objectives Google Colab / Replit Git basics Code documentation
🛠
Project: Build a student grade calculator in Python — input scores, calculate average, print result with pass/fail.
By end of month 3: Write and run real Python programs. Think logically. Opens doors to AI, data, web, and automation.
M4
Data & Cloud

Month 4 — Databases & cloud fundamentals

Aligned with: Oracle SQL · AWS Cloud Practitioner (CLF-C02)

▾

Core topics

What is a database? Tables, rows, columns SQL: SELECT, INSERT, UPDATE Joins & relationships What is cloud computing? IaaS / PaaS / SaaS AWS / Azure / Google Cloud Cloud storage & security

Certification aligned

AWS Cloud Practitioner MySQL Workbench Data privacy & GDPR Spreadsheet vs database
🛠
Project: Create a student records database in SQLite — store names, grades, courses. Query and filter with SQL commands.
By end of month 4: Store, retrieve, and manage real data. Understand how apps like Gmail and Facebook store information. Cloud concepts ready for interviews.
M5
AI & ML

Month 5 — AI concepts & machine learning literacy

Aligned with: Google AI Essentials · DeepLearning.AI · UNESCO AI curriculum

▾

Core topics

What is AI / ML / DL? How models are trained Supervised vs unsupervised Large language models Neural networks (visual) AI bias & ethics AI in Afghan daily life When NOT to trust AI

Certification aligned

Google AI Essentials cert Prompt engineering basics AI tools comparison AI in the job market
🛠
Project: Use Claude or ChatGPT to build a Pashto study assistant — write prompts, test outputs, document what works and what fails.
By end of month 5: Explain AI to anyone. Use it wisely. Understand its limits. Think with AI — not instead of thinking.
M6
Career ready

Month 6 — AI tools mastery & career launch

Aligned with: Microsoft AI Skills · LinkedIn Learning · Freelance readiness

▾

Core topics

ChatGPT, Claude, Gemini Image AI (Midjourney) AI for productivity Free vs paid tools Building a portfolio Tech CV writing Upwork / Fiverr basics Remote job platforms

Certification aligned

Microsoft AI Skills badge LinkedIn optimization GitHub portfolio setup Interview preparation
🏆
Capstone project: Build a personal portfolio page using HTML + CSS + AI tools. Write your CV. Apply to 3 real jobs or freelance gigs.
By end of month 6: Job-ready. Portfolio on GitHub. CV written. First applications submitted. Ready to pursue CompTIA A+, Network+, or AWS certifications.

How to study

Study strategies that actually work

No expensive courses. No perfect internet. No full-time schedule. Just methods proven to work for self-taught tech learners.

01

Protect your study time

30 focused minutes daily beats 5 inconsistent hours a week. Block the same time every day — even if it's early morning or late at night.

02

Learn by doing, not watching

For every 10 minutes you watch or read, spend 20 minutes trying it yourself. Mistakes are the fastest way to actually learn.

03

Study offline too

Download lessons, PDFs, and videos when you have internet. Use offline tools like Ollama for AI practice without a connection.

04

Teach what you learn

Explain each topic to someone else in Pashto — a family member, a friend, or even yourself on video. If you can't explain it, you don't know it yet.

05

Use spaced repetition

Review yesterday's notes before starting today's new material. The brain remembers things it revisits — not things it sees once.

06

Build something every month

Each month ends with a project. Don't skip it. A portfolio of 6 real projects proves your skills far more than certificates alone.
